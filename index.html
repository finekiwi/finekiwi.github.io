About

I'm an AI researcher focused on making complex information accessible through intelligent systems. My work spans RAG-based chatbots for welfare policy navigation, computational analysis of how visual features affect user perception, and computer vision systems for industrial applications.

I hold an M.S. in AI Convergence from Ewha Womans University (4 publications, 2 patents). My research explores how to design AI that bridges the gap between technical capabilities and real user needs—whether that's a citizen searching for welfare benefits or an inspector reviewing manufacturing defects.

Projects

Welfare Compass

Seoul AI Hackathon · Top 20 / 181 teams · 2025

Problem

300+ welfare policies scattered across documents — citizens struggle to determine eligibility, especially those with low digital literacy.

Approach

  • RAG pipeline using LangChain + ChromaDB for policy document retrieval
  • Natural language query processing to extract relevant policies and eligibility criteria
  • Streamlit-based conversational interface for accessible interaction

Outcome

Demonstrated effective retrieval for complex policy queries, advancing to hackathon finals.

Tech Stack

Python LangChain ChromaDB GPT API Streamlit

HCI Relevance: Lowering language barriers to improve information accessibility for non-expert users navigating complex bureaucratic systems.

Drama Color Pattern Analysis

Published in Journal of the Korean Data Analysis Society · Feb 2025

Problem

Color's impact on viewer emotional engagement was based on subjective judgment — no quantitative analysis existed for cross-cultural comparison.

Approach

  • ConvNeXtLarge for scene clustering across drama episodes
  • HSV/LAB color metrics comparing Korean vs Spanish versions of Money Heist
  • User survey (n=86) for preference validation

Outcome

Korean color grading preferred 1.5× regardless of viewer nationality (p<.05). First author publication.

Tech Stack

Python PyTorch ConvNeXtLarge CIELAB Analysis

HCI Relevance: Quantifying how visual features affect user perception and emotion — applicable to interface design color decisions.

Welding Defect Detection

Presented at IPIU 2024

Problem

No real defect images available in manufacturing environments due to security and cost constraints — severe training data shortage.

Approach

  • CycleGAN for domain adaptation (X-ray radiographs → inspection images)
  • Preprocessing pipeline (contrast enhancement, wavelet filtering) to suppress artifacts
  • YOLOv5 with two-stage transfer learning (COCO → synthetic defects)

Outcome

87.5% mAP achieved on synthetic-to-real defect detection. First author conference paper.

Tech Stack

Python PyTorch CycleGAN YOLOv5

HCI Relevance: Transforming expert workflow from "review thousands of images manually" to "query similar defects" — improving inspector efficiency and reducing cognitive load.
